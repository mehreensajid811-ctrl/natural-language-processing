Project Name  : Natural Language Processing Library
Topic         : Custom C++ Library - Text and Language Processing

This project implements a custom C++ library for natural language 
processing tasks. The Expense class manages and tracks language 
processing costs and operations associated with different users. 
It demonstrates function overloading through multiple versions of 
setExpense() allowing different combinations of person and amount 
parameters. Operator overloading is implemented for arithmetic 
(+, -, *), comparison (==, <), and output (<<) operations to 
enable easy manipulation and display of processing data.
